# Digital Art Gallery Website

This project is a colorful, mobile-optimized digital art gallery. It features:
- Home page with gallery name and navigation
- About Me page with 4 scroll-snapping panels (artist photos and bios)
- Gallery page with scrolling grid of grayscale thumbnails (color on hover/tap, popup for full image)
- Sticky back-to-top button
- Responsive design for mobile and desktop
- Easy image management: add images to the `gallery` folder for automatic display

## Getting Started
1. Place your gallery images in the `gallery` folder (e.g., gallery1.jpg, gallery2.png, etc.).
2. Place your About Me images as aboutme1.jpg, aboutme2.jpg, etc.
3. Open `index.html` to view the site.

## Features
- Grayscale-to-color transitions for images and panels
- Popup modal for full-size image viewing (maintains aspect ratio, no distortion)
- Scroll-snapping About Me panels (centered, with partial previews of adjacent panels)
- Mobile-first, fully responsive layout
